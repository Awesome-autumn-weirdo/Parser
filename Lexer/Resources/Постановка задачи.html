Тип record в Pascal

Запись – это структура данных, состоящая из фиксированного числа компонент, называемых полями. Каждое поле может иметь свой тип данных.

Общий вид описания типа record в языке программирования Pascal:

type t = record
    id11, id12, …: type1;
    id21, id22, …: type2;
    ...
end;

Здесь id – идентификаторы полей; type – типы полей; t – имя типа.

Примеры

Пример 1. Данные комплексного типа

Можно описать переменной типа record.

type complex = record
    re, im: real
end;

Здесь complex – имя типа. Переменная состоит из двух полей: re и im, имеющих один и тот же тип (real).

Пример 2. Даты событий

Можно описать следующим образом:

type date = record
    month: 1..12;
    day: 1..31;
    year: integer
end;

В этом примере описан тип date. Запись состоит из трех полей: month, day и year. Каждое поле содержит соответственно данные: целое число в пределах от 1 до 12 (номер месяца), целое число от 1 до 31 (день), целое число (год).
