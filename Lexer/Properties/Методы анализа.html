Диаграмма состояний грамматики G[OPR]

Рис. 1. Диаграмма состояний для грамматики G[OPR]

На рис. 1 приведена диаграмма состояний для грамматики G[OPR]. На диаграмме представлена посимвольная декомпозиция записей с генерацией соответствующего символического кода символов <идентификатор>, <ключевое слово> и литер «:», «=» и др.

Непомеченные дуги на диаграмме соответствуют состоянию ERROR (отсутствие данного символа в словаре грамматики) либо выходу из обработки очередного символа и переходу на start для обработки следующего терминального символа.

Особенности диаграммы:

  • Состояния отображают процесс разбора записи Pascal
  • Переходы между состояниями маркированы терминальными символами
  • Специальные символы (:, =, ,) выделены отдельными переходами
  • Обработка ошибок реализована через состояние ERROR
