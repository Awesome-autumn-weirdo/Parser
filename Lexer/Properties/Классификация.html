Анализ автоматной грамматики (тип 4 по Хомскому)

Грамматика относится к автоматным грамматикам (тип 4 по Хомскому). Разберём каждое правило и проверим его соответствие автоматной грамматике:

1)
<OPR> → 'type' <SPACE>
A→aB (праворекурсивные продукции)
2)
<SPACE> → ' ' <ID>
A→aB (праворекурсивные продукции)
3)
<ID> → letter <IDREM>
A→aB (праворекурсивные продукции)
4)
<IDREM> → letter | digit <IDREM>
A→aB (праворекурсивные продукции)
5)
<IDREM> → '=' <REC>
A→aB (праворекурсивные продукции)
6)
<REC> → 'record' <SPACE>
A→aB (праворекурсивные продукции)
7)
<SPACE> → ' ' <FIELD>
A→aB (праворекурсивные продукции)
8)
<FIELD> → letter <FIELDREM>
A→aB (праворекурсивные продукции)
9)
<FIELDREM> → letter | digit <FIELDREM>
A→aB (праворекурсивные продукции)
10)
<FIELDREM> → ',' <FIELDREM>
A→aB (праворекурсивные продукции)
11)
<FIELDREM> → ':' <TYPE>
A→aB (праворекурсивные продукции)
12)
<TYPE> → type <END>
A→aB (праворекурсивные продукции)
13)
<END> → 'end' <SEMICOLON>
A→aB (праворекурсивные продукции)
14)
<SEMICOLON> → ';'
A → a (терминальные продукции)

Вывод:

Грамматика является автоматной (тип 4): Все правила либо праворекурсивные (A → aB), либо терминальные (A → a).
